🐳 Docker Compose Project

Многоконтейнерное приложение с Node.js, Redis и PostgreSQL

🟢

Node.js + Express

Веб-сервер, обрабатывающий HTTP-запросы и предоставляющий API для взаимодействия с базами данных.

🔴

Redis

In-memory хранилище для кэширования данных и обеспечения быстрого доступа к часто используемой информации.

🐘

PostgreSQL

Надёжная реляционная база данных для постоянного хранения структурированных данных приложения.
